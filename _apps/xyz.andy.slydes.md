This fun slideshow app uses markdown to descibe a full presentation in a single file.
It supports the formatting of standard markdown and can edit in real time as well as present from the file.

Content is layed out according to what is in each slide.
Code is automatically formatted and syntax highlighting added.
Themes are supported and you can save/load your presentation and export to PDF.
